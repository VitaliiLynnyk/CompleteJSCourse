Coding challenge 4

Task

remember Mark and John BMI Challenge. Implement the same functionality with objects and methods.
1. For each of them, create an object with properties for their full name, mass, and Height
2. Then, add a method to each object to calculate the BMI. Save the BMI to the object and also return it from the method.
3. In the end, log to the console who has the highest BMI, together with the full name and the respective BMI. Don't forget they might have the same BMI. Remember: BMI = mass / heightË†2 = mass / (height * height). (mass in kg and height in meter).

Code 

          class Person {
            constructor(fullName, mass, height) {
              this.fullName = fullName;
              this.mass = mass;
              this.height = height;
            }
          
            getFullName() {
              return this.fullName;
            }
          
            getBMI() {
              return +this.mass / Math.pow(+this.height, 2);
            }
          
            compareBMI(person) {
              if (person instanceof Person) {
                return this.getBMI() === person.getBMI()
                  ? `${this.getFullName()} has the same BMI as ${person.getFullName()}`
                  : this.getBMI() > person.getBMI()
                    ? `${this.getFullName()} has a higher BMI as ${person.getFullName()}`
                    : `${person.getFullName()} has a higher BMI as ${this.getFullName()}`;
              }
            }
          }
          
          const mark = new Person('Mark Cage', 78, 1.75);
          const john = new Person('John Smith', '82', 1.65);
