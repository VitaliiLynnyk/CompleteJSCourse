Task 1

BMI: mass / height^2 = mass / (height * height)
1.store mark and john's mass and height in variables.
2.calculate both their bmis.
3.create a boolean variable containing information about whether Mark has a higher BMI than John
4.Print a string to the console containing the variable from Step 3. (Something like "Is Mark's BMI higher than John's? true").

Code 

        const Mark = {
          mass: "50kg",
          height: "37cm"
        };
        
        const John = {
          mass: "100kg",
          height: "195cm"
        };

        const calcBmi = person => {
          const mass = parseInt(person.mass.slice(0, person.mass.length - 2));
          const height = parseInt(person.height.slice(0, person.height.length - 2));
          return mass / height * height;
        }

        const markBMI = calcBmi(Mark);
        const JohnBMI = calcBmi(John);
        const res = markBMI > JohnBMI
                    ? `Mark has ${markBMI} grater BMI then John ${JohnBMI}`
                    : `John ${JohnBMI} has grater BMI then Mark ${markBMI}`;
        console.log(res);
